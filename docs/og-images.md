# OG image copy

Text for per-page Open Graph images (1200×630). Constant elements on every
image, mirroring the app's nameplate-on-canvas hero: paper-grain canvas, the app
icon + Compress Pro wordmark top-left, and four Geist Mono uppercase
chips (`COMPRESS-PRO.COM` `FREE` `PRIVATE` `NO ADS`) along the bottom. Headline large
(Plus Jakarta Sans, same weight and tracking as the hero), subline smaller and
muted — monochrome throughout, like the site.

Images are generated by `pnpm og` (`scripts/generate-og.mjs` — Playwright
renders an HTML template, sharp downscales to 1200×630 JPEG). The script
mirrors this table; edit copy here, then re-run it. Output: `static/og.jpg`
(default) + `static/og/<name>.jpg`, wired as `ogImage: '/og/<name>.jpg'` on
each entry in `src/lib/seo.ts` — pages without one fall back to `/og.jpg`.
